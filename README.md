## About the Repository
This repository contains the project files for HW #4.3, the Reproducible Data Analysis Document. Included files are a plan document showcasing the goal, needs, and steps taken to complete the project, a .qmd file which contains the data cleaning, visualizations, and narratives of the project, and the output file which is a .pdf of the rendered .qmd file.

## Data
Data used in the "Busiest Airports" section of the project comes from the passenger traffic data tables on Wikipedia articles about the airports: Hartsfield-Jackson Atlanta Airport, Frankfurt Airport, Beijing Daxing Airport, Incheon Airport, Shanghai Pudong Airport, and Dubai Airport. 

Data used in the "Planning and Prompting GenAI" section of the project comes from a .csv file provided by the instructors of the course.

## Plan
The plan for the project is to produce a document in which another data analyst or data scientist can read the contents of the document like the code chunks, visualiaztions, and narratives, and be able to reproduce, or improve upon the outcomes.

## Organization
This repository contains four files. The current document being the README.md which has a brief overview of the project contents and purpose. The rep_data_analysis.qmd file which is the Quarto markdown document that renders into a finalized document. The rep_data_analysis.pdf document which is the rendered and finalized quarto document. The plan.md which contains the step-by-step plans, goals, and needs of completing the projecet and managing the repository.
